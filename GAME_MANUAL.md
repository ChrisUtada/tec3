# TEC3 游戏手册

## 📖 游戏概述

《异动区域已经锁定》是一款基于卡牌的叙事推理游戏。玩家扮演一名调查员，通过探索场景、收集线索、与NPC对话来揭开一个被异常力量扭曲的世界背后的真相。

---

## 🎮 核心玩法

### 1. 卡牌操作

#### 拖拽与放置
- **拖放卡牌**：点击并拖动卡牌到目标位置
- **双击卡牌**：打开对应功能（人物对话、场景探索、归因推演）
- **卡牌堆叠**：将卡牌拖放到另一张卡牌上可形成堆叠

#### 功能卡牌
| 卡牌 | 功能 |
|------|------|
| 🔍 **观测** | 拖放到任意卡牌上查看详细信息 |
| 🎯 **捕获** | 放置在人物/场景/物品上获取隐藏线索 |
| ⚡ **逻辑归因** | 双击打开归因推演界面 |
| ⚰️ **献祭** | 将卡牌拖放到此卡上转化为因果律 |

### 2. 场景探索

将**初级调查员卡**拖入场景卡的槽位中进行探索：

```
┌─────────────────┐
│  植物学家的花园  │
├───────┬───────┬───────┤
│ 槽位1 │ 槽位2 │ 槽位3 │
│[调查员]│       │       │
└───────┴───────┴───────┘
```

**探索结果**：
- 获取物品和线索卡牌
- 有概率获得**疲劳卡**（负面状态）
- 疲劳卡达到5张时无法进行探索、对话和归因

### 3. 人物对话

双击人物卡牌打开对话面板：
- 将物品/场景/线索卡拖入槽位触发对话
- 不同卡牌组合会触发不同的对话内容
- 对话可能揭示关键剧情线索

### 4. 归因推演

双击**逻辑归因卡**打开推演界面：
- 将收集的线索拖入槽位进行组合
- 不同线索组合揭示不同的真相
- 正确的线索组合可触发结局

### 5. 休息系统

双击**休息处**卡牌：
- 放入**初级调查员**和**疲劳卡**
- 消耗疲劳卡恢复调查员状态

### 6. 因果律系统

- 通过**献祭卡牌**获取因果律
- 因果律可用于改变事件概率或购买特殊物品

## 📜 剧情背景

### 故事开端

你是一名新加入 TEC（调查员联合机构）的初级调查员。你的第一个任务是调查一位失踪的植物学家——陆珩松。根据线索，他最后的踪迹指向了一个被称为"植物学家的花园"的神秘地方。

### 探索路径

```
TEC总部 ──(调查员)──▶ 揭示TEC真身
    │
    ▼
植物学家的花园 ──(探索)──▶ 发现书桌、唱机、黑影等
    │
    ▼
书桌 ──(调查)──▶ 发现笔记、抽屉
    │
    ▼
抽屉 ──(调查)──▶ 手电筒、剪刀、木盒
```

### 核心谜团

1. **TEC 的真实身份**：这个自称"调查员联合机构"的存在到底是什么？
2. **植物学家的失踪**：陆珩松在花园里发现了什么导致他失踪？
3. **黑影的身份**：花园中闪过的黑色人影想传达什么？
4. **真名的秘密**：被揭示的"深渊低语者"究竟是什么？

## 🏆 结局系统

根据投入归因推演仪的线索组合，可触发不同结局：

| 结局 | 所需线索 | 描述 |
|------|----------|------|
| 🌑 吞噬废墟 | 铁锈的味道 | 世界污染无法挽回，选择品尝终极苦涩 |
| 📖 宗族之名 | 宗族谱本 | 揭示家族消亡的历史真相 |
| 👁️ 虚假现实 | 信号屏蔽 + 墙上的阴影 | 陷入视觉迷宫，质疑现实 |
| 🏛️ 深渊低语者 | 被揭示的真名 | 念出旧日支配者的真名，梦境破碎 |
| 🌀 完全觉醒 | 铁锈的味道 + 宗族谱本 + 信号屏蔽 | 三重验证完全觉醒，真相大白 |

## ⚠️ 游戏提示

1. **合理管理疲劳**：及时使用休息处恢复状态，避免疲劳堆积
2. **多尝试组合**：不同的卡牌组合可能触发意想不到的剧情
3. **仔细阅读描述**：每张卡牌的描述都包含重要线索
4. **探索所有场景**：不要错过任何可能的线索来源
5. **保存因果律**：关键时刻因果律可能改变游戏走向
