# 🎮 TEC3 - 卡牌探索游戏

一款基于Web的卡牌探索游戏，通过堆叠组合和场景探索揭示故事真相。

## 🚀 快速开始

### 启动开发服务器

```bash
# 进入项目目录
cd TEC3

# 启动本地服务器
node server.js

# 打开浏览器访问
http://localhost:8080
```

### 端口冲突处理

如果端口8080被占用：

```bash
# 查找占用进程（Windows）
netstat -ano | findstr :8080

# 终止进程（替换PID为实际进程号）
taskkill /F /PID <PID>

# 重新启动服务器
node server.js
```

## 🎲 游戏操作指南

### 基础操作
- **拖拽**：点击卡牌并拖动到目标位置
- **堆叠**：将卡牌拖放到另一张卡牌上进行组合（可堆叠多张卡，需要根据具体内容进行设定）
- **双击**：双击卡牌查看详情或触发特殊效果

### 探索流程
1. 点击场景卡打开探索节目，场景卡不可放入槽位中探索
2. 在探索节目的槽位中放入人物卡（必须）和其他卡牌（可选）
3. 点击"探索"按钮
4. 等待进度条完成
5. 自动获得掉落物

### 组合规则
- 将至少两张卡牌（可以是若干张卡牌）拖放到一起即可触发组合
- 顺序无关，A+B 与 B+A 效果相同
- 组合成功会显示进度条和提示信息

### 回收机制
- 将卡牌拖放到"物资回收站"可兑换金币
- 每张卡牌价值10金币
- 金币的作用，可以提升掉落概率以及购买一些特殊物品

## 📝 配置说明

### 添加新卡牌
编辑 `js/config.js` 中的 `CARD_TEMPLATES` 对象：

```javascript
'NEW_CARD_ID': {
    name: '卡牌名称',
    desc: '卡牌描述',
    type: 'item',      // 类型：action/item/scene/char/clue/logic
    class: 'item-card', // CSS类名
    consumable: true    // 是否可消耗
}
```

### 添加新组合
编辑 `js/config.js` 中的 `CARD_COMBINATIONS` 对象：

```javascript
'CARD1+CARD2': {
    type: 'interaction',      // 组合类型
    result: ['RESULT_CARD'],  // 产出卡牌
    message: '组合提示信息',
    delay: 2000,             // 动画时长（毫秒）
    consumeAll: false         // 是否消耗全部材料
}
```

### 添加新场景
编辑 `js/config.js` 中的 `SCENE_EXPLORATION` 对象：

```javascript
'SCENE_ID': {
    name: '场景名称',
    slots: 3,                 // 槽位数
    exploreTime: 5000,        // 探索时间（毫秒）
    message: '探索提示',
    drops: [                  // 掉落配置
        { templateId: 'CARD_ID', chance: 0.6, message: '掉落提示' }
    ]
}
```
